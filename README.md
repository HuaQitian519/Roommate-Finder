# Roommate-Finder

一款帮助同学寻找室友的开源项目，基于 **Python** 和 **Flask** 框架开发。通过向量化特征快速匹配室友，为同学们提供更方便的室友匹配服务！

## 项目特点

- **开源项目**：完全开源，欢迎贡献代码或提出建议。
- **快速匹配**：基于向量化特征，快速找到合适的室友。
- **简单部署**：支持 Docker 一键部署，省时省力。
